Setup Instructions

  1. Get OpenWeatherMap API Key:
    • Go to OpenWeatherMap API
    • Sign up for a free account
    • Get your API key from "My API Keys" section
  2. Configure Environment:
    # Create .env file in frontend directory
    VITE_OPENWEATHER_API_KEY=your_api_key_here
    VITE_API_URL=http://localhost:8000
  3. Start the Application:
    cd frontend
    npm run dev
  4. View the Dashboard:
    Open http://localhost:5173 in your browser
    Navigate to the Dashboard page
    Look for "Live Weather & Storm Map" section

Features Included

  • Real-time Weather Data: Temperature, humidity, pressure for major Indian cities
  • Wind Visualization: Speed, direction, and animated indicators
  • Storm Tracking: Simulated cyclone data with path prediction
  • Interactive Controls: Layer toggles, refresh button, legend
  • Heat Map Overlay: Temperature visualization across regions
  • Responsive Design: Works on desktop and mobile devices
